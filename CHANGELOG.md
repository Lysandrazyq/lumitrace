## 2026-06-14

### 官网 · 数据备份与恢复（v2.5.0）
- 页脚新增「数据备份与恢复」：一键把所有应用的本地数据导出成 JSON 文件，换设备 / 清缓存前先备份。
- 导入备份即可一次性恢复全部应用数据（心迹 / 恒迹 / 书迹 / 一隅 / 时记…）。
- 纯本地，备份文件只存在你自己的设备，不上传。

### 全线 · 离线与可安装（Service Worker + PWA）
- **离线访问**：所有应用装上 Service Worker，断网也能打开用过的应用（资源缓存优先、入口页网络优先）。
- **添加到主屏幕**：塔罗牌 / 墨迹 / 片刻 / 时记 / 一隅 / 心迹 / 书迹 / 恒迹 全部补齐 `manifest`，可作为独立窗口应用安装，带品牌色启动屏与图标。

### 恒迹 v1.0.0（新应用）
- 习惯打卡：多习惯管理、🔥 连续天数、当前/最佳连续统计。
